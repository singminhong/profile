Exhibit Presentation System (EPS)

Real-time display of evidence and annotation of image during a court case.

Below is a testing video during software development.

Exhibit Presentation System (EPS) consists of a Controller screen and Editor screen. The interpreter uses the Controller screen to:

  • 1. Select court case, exhibits and retrieve evidence images
  • 2. Capture or open evidence image
  • 3. Create new or clone exhibits
  • 4. Select Editor user, Editor annotation color and drawing ball size
  • 5. Manipulate and extend evidence image to Editor Screen
  • 6. Control over Editor Screen
  • 7. Export annotated evidence image to PDF locally or back to EPS database

The defendant and witnesses use the Editor screen to annotate extended image.

  • Date: January 2019
  • Client: The High Court in Sabah and Sarawak
  • Category: Desktop Application. Windows Application. IPEVO Visualizer. JSON. C#.

Queue Management System (QMS)

Provide and cater ticketing for queue and counter services.

Above is our QMS been used by one of our clients - Padawan Municipal Council.

Queue Management System (QMS) consist of softwares KIOSK, COUNTER, PUBLIC DISPLAY and RATING.

  • KIOSK - A standalone machine that allow citizen to select service and get queue ticket.
  • COUNTER - For government workers to CALL, CANCEL, PASS, WAIT, RECALL and END ticket from citizen.
  • PUBLIC DISPLAY - To display current and past ticket numbers and counters.
  • RATING - For citizen to rate the government worker's services after the service ends.
  • Date: April 2017
  • Clients: 1. Padawan Municipal Council 2. Kuching Resident and District Office
  • Category: Desktop Application. Windows Application. C#. MYSQL. Windows Services. Web Services.

Wireless Gateway Module for Safety Helmet

Wirelessly monitor worker's condition working in confined-space.

Above is the project overview.

Above is the software look.

Above is the award and recognition received for this project.

Above is the demostration of the project.

Wireless Gateway Module for Safety Helmet is a wireless monitoring system aim to monitor worker working in confined space and unreachable environment. The system consists of a software and a worker helmet. It live-streams and monitors heart rate, SPO2, relative humidity and temperature via IEEE 802.11 WiFi.

  • Date: June 2016
  • Scope: Undergraduate Studies Final Year Project
  • Category: Desktop application. Windows application. Python. Raspberry Pi. SPO2 Sensor. PiCamera module. Humidity sensor DHT11.
